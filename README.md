# Ascendia

**Ascendia** is a Progressive Web App (PWA) built to support Cambridge International students in preparing for their O Level, IGCSE, AS, and A Level exams. It offers a cross-platform, installable experience with tools for syllabus tracking, past paper organization, performance analytics, exam countdowns, and custom study planning.

This repository contains the frontend source code for the Ascendia application. It is made publicly viewable for demonstration and portfolio purposes only.

---

## Purpose

Ascendia was developed to solve common problems students face during exam preparation:

- Difficulty tracking syllabus coverage for multiple subjects
- Disorganized past paper practice and lack of performance insight
- Inconsistent or inefficient study planning
- No centralized space to manage all academic materials

This project aims to centralize the exam preparation process into a modern, clean, and user-friendly digital platform accessible from any device.

---

## Features

### 1. Syllabus Checklist
- Select subjects based on official Cambridge syllabuses
- View and manage each topic as a checklist
- Progress is saved locally to persist across sessions

### 2. Past Paper Vault
- Upload and organize past papers by subject and year
- Tag status: Unattempted, Attempted, Reviewed
- Add custom notes or reflections for each paper

### 3. Mark & Performance Tracker
- Enter scores from past paper attempts
- View average performance and trends over time
- Automatically calculates grade ranges based on mark schemes

### 4. Exam Countdown
- Input upcoming exam dates
- See real-time countdowns for each subject on the dashboard
- Visual exam calendar to manage time effectively

### 5. Study Planner & Timetable Generator
- Create a daily/weekly study timetable
- Allocate specific subjects and topics per session
- Editable templates to match personal productivity style
